# Privacy Policy for Conversation Assistant

**Last Updated: November 24, 2024**

## Overview

Conversation Assistant is a macOS application that provides real-time AI-powered assistance during technical conversations, with note-taking capabilities and code analysis. This privacy policy explains how we handle your data.

## Data Collection

Conversation Assistant collects and processes the following data:

### 1. Screenshots
- **What we collect:** When you use the Coding Task feature, the app captures screenshots of your screen
- **How it's used:** Screenshots are sent to Anthropic's Claude AI API for analysis and code assistance
- **Storage:** Screenshots are stored temporarily in memory during your session only. They are NOT saved to disk and are cleared when you close the app or clear the session
- **Third-party sharing:** Screenshots are transmitted to Anthropic Inc. for AI analysis via their Claude API

### 2. Notes Content
- **What we collect:** Text notes you write in the Notes tab
- **How it's used:** For your personal reference during conversations
- **Storage:** Notes are stored locally on your device only
- **Third-party sharing:** Notes are NEVER transmitted to any external service

### 3. API Keys
- **What we collect:** Your Anthropic API key for Claude AI access
- **How it's used:** To authenticate your requests to Anthropic's API
- **Storage:** API keys are stored in memory only during the app session. They are NOT saved to disk or system keychain
- **Third-party sharing:** Your API key is sent directly to Anthropic's servers with each API request (standard API authentication)

## Third-Party Services

### Anthropic Claude AI
- **Service Used:** Claude Haiku 4.5 via Anthropic API
- **Data Shared:** Screenshots and prompt text for code analysis
- **Purpose:** AI-powered code analysis and problem-solving assistance
- **Privacy Policy:** [https://www.anthropic.com/legal/privacy](https://www.anthropic.com/legal/privacy)
- **Data Processing:** Anthropic may process screenshots temporarily to generate responses. Refer to Anthropic's privacy policy for details on their data handling practices

## Data Retention

- **Screenshots:** Deleted immediately after analysis or when you clear the session (⌘G)
- **Notes:** Remain on your device until you manually delete them
- **API Keys:** Cleared when the app quits
- **No cloud storage:** We do not maintain any cloud storage or servers. All data remains on your device

## Data Security

- **App Sandbox:** Conversation Assistant runs in Apple's App Sandbox for enhanced security
- **No telemetry:** We do not collect analytics, crash reports, or usage statistics
- **No tracking:** We do not track your behavior or identify you in any way
- **Local storage only:** All persistent data is stored locally on your Mac

## Your Data Rights

Since all data is stored locally on your device:
- **Access:** You have full access to all your data (notes stored locally)
- **Deletion:** Delete any data by clearing the session (⌘G) or uninstalling the app
- **Control:** You control when screenshots are captured and when analysis is performed

## Screen Capture Permission

Conversation Assistant requires screen capture permission to take screenshots of coding problems. This permission is:
- **Required by macOS:** You will see a system permission dialog on first use
- **User-controlled:** You can revoke this permission anytime in System Settings → Privacy & Security → Screen Recording
- **Purpose:** Only used when you explicitly press ⌘S to capture a screenshot

## Children's Privacy

Conversation Assistant is not directed to children under 13. We do not knowingly collect data from children.

## Changes to This Policy

We may update this privacy policy from time to time. The "Last Updated" date at the top will reflect when changes were made. Continued use of the app after changes constitutes acceptance of the updated policy.

## Legal Basis (GDPR)

For users in the EU/EEA, our legal basis for processing data is:
- **Legitimate Interest:** Processing screenshots for AI analysis based on your explicit request
- **Consent:** Using your API key with your explicit consent and configuration

## California Privacy Rights (CCPA)

California residents have the right to:
- Know what personal information is collected
- Delete personal information (by clearing app data)
- Opt-out of sale of personal information (we do not sell data)

## Summary

**In plain English:**
- Your notes stay on your Mac only
- Screenshots are sent to Anthropic AI only when you click "Analyze"
- Your API key is stored in memory, not saved anywhere
- We don't collect any data about you or track your usage
- Uninstall the app to delete all data
