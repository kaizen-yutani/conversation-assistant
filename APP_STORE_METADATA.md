## Description (4000 characters max)

```
Interview Master - Your Private AI Interview Assistant

Ace your technical interviews with a secure, screen-share-invisible note-taking app powered by Claude AI.

🔒 INVISIBLE TO SCREEN SHARING
Interview Master is designed to be hidden from Zoom, Teams, and all screen sharing apps. Your notes and AI assistance remain completely private during virtual interviews.

📝 TWO POWERFUL MODES

Interview Notes Tab:
• Quick-access interview prep notes and answers
• Markdown formatting with syntax highlighting for code
• Instant search functionality (⌘F) - find anything in seconds
• Searchable knowledge base for common questions
• Beautiful glass UI that's easy to read

Coding Task Tab:
• Capture screenshots of coding problems (⌘S)
• AI-powered analysis with Claude Haiku 4.5
• Two intelligent analysis modes:
  - Problem Solving: Get approach, solution, and time/space complexity analysis
  - Code Review: Find bugs, security issues, and improvement suggestions
• Store up to 7 screenshots per session
• Streaming AI responses for fast feedback

⌨️ KEYBOARD SHORTCUTS FOR SPEED
• ⌘L - Instantly hide/show window
• ⌘1/⌘2 - Switch between Notes and Coding tabs
• ⌘S - Capture screenshot (Coding tab)
• ⌘Enter - Analyze with AI
• ⌘F - Search your notes
• ⌘M - Toggle analysis mode
• ⌘G - Clear screenshots

🎯 PERFECT FOR:
• Software engineering interviews
• Coding assessments and take-home challenges
• Technical phone screens
• Live coding challenges on Zoom/Teams
• System design discussions
• Algorithm practice and preparation

💡 KEY FEATURES:
• Claude Haiku 4.5 AI integration for fast, accurate analysis
• Markdown editor with syntax highlighting
• Screenshot capture without interrupting your workflow
• Two analysis modes: Problem Solving & Code Review
• Always-on-top window mode
• Keyboard-driven for efficiency
• Privacy-focused: all notes stored locally only

🛡️ PRIVACY & SECURITY:
• All notes data stored locally on your Mac
• API keys stored in memory only (never saved to disk)
• No telemetry or tracking
• No accounts or sign-ups required
• Screen share exclusion built-in
• Screenshots sent to Anthropic API only when you click "Analyze"

🔧 REQUIREMENTS:
• Anthropic API key for AI analysis (usage-based pricing from Anthropic)
• macOS 14.0 (Sonoma) or later
• Internet connection for AI features

Interview Master gives you the confidence of having your best answers at your fingertips, without compromising interview integrity. Study hard, prepare well, and let Interview Master be your safety net during the big moment.
```
